# Introduction

Fastify is a web framework highly focused on providing the best developer
experience with the least overhead and a powerful plugin architecture. It is
inspired by Hapi and Express and, as far as we know, it is one of the fastest
web frameworks in town.

## Why another framework?

An efficient server implies a lower cost of infrastructure, better
responsiveness under load, and happy users. Fastify helps you handle your
server resources efficiently — serving the highest number of requests possible
without sacrificing security validations or developer ergonomics.

## Core principles

- **Performance first** — up to ~30k requests per second depending on complexity.
- **Schema based** — validate and serialize with JSON Schema, compiled to fast functions.
- **Encapsulated plugins** — isolation by default keeps large apps maintainable.
- **TypeScript ready** — first-class type declarations shipped with the framework.
